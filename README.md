## Hello Shapes

#### Getting Started

```shell
$ cd hello-shapes
$ npm install -g gulp           # Install Gulp task runner globally
$ npm install                   # Install Node.js components listed in ./package.json
```

#### How to clean

```shell
$ gulp build                    # or, `npm run build`
```

#### How to build

```shell
$ gulp build                    # or, `npm run build`
```

#### How to lint
```
$ npm run lint
```

#### How to generate API documentation
```
$ gulp docs
```

#### How to test
```
$ npm test
```

#### How to run

```shell
$ npm start
```

#### How to run with mock data
```
$ npm run mock
```
